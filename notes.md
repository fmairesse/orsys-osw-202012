## Outillage

- installer Node avec avec NVM : rechercher "NVM windows"
- quelques commandes:
  - `npm install`: récupère toutes les dépendances dans un dossier `node_modules`
  - `npm install lodash`: installe la nouvelle dépendance
- `package.json`
  - décrit les dépendances du projet
  - permet le partage de scripts avec les collègues
    - les scripts ont accès aux outils stockés dans `node_modules`
    - sont lancés avec `npm run monscript`
